## üß© Component Usage

### Buttons

```kotlin
// Primary Gradient Button
GradientButton(
    text = "Submit",
    onClick = { /* action */ }
)

// Icon Button with Glow
IconButtonWithGlow(
    icon = Icons.Filled.Settings,
    contentDescription = "Settings",
    onClick = { /* action */ }
)

// Mic Button (Floating)
PulsingMicButton(
    onClick = { /* start recording */ },
    isListening = false
)
```

### Cards

```kotlin
// Neon Card (Standard)
NeonCard {
    Text("Card Content")
}

// Stat Card

    title = "Income",
    value = "‚Çπ10,000",
```

## üß≠ Navigation Commands

### Navigate to Screen

```kotlin
navController.navigate(Screen.Home.route)
```

### Navigate Back

```kotlin
navController.popBackStack()
```

### Navigate with Pop

```kotlin
navController.navigate(Screen.Home.route) {
    popUpTo(Screen.Welcome.route) { inclusive = true }
}
```

### Bottom Navigation

```kotlin
AppBottomNavigation(
    currentRoute = currentRoute,
    onNavigate = { route -> 
        navController.navigate(route) 
    }
)
```

## üé≠ Animation Specs

```kotlin

    initialValue = 1f,
    targetValue = 1.05f,
    animationSpec = infiniteRepeatable(
        animation = tween(2000),
        repeatMode = RepeatMode.Reverse
    )
)
```

```kotlin
AnimatedVisibility(
    visible = isVisible,
    enter = fadeIn() + slideInVertically(),
```

## üéØ Key Interactions

### Voice Input

```
User taps mic button ‚Üí VoiceService.startListening()
‚Üí Transcription received ‚Üí ViewModel.processTranscription()
‚Üí UI updates with response
```

### Transaction Actions

```
User taps transaction ‚Üí Dialog appears
‚Üí Options: Read Aloud / Modify / Delete
‚Üí Action executed via ViewModel
```

### Bottom Navigation

```
User taps tab ‚Üí Navigate to screen
‚Üí Previous state restored if available
‚Üí Smooth transition with animations
```
